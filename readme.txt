----------------------------------------------------------------TIPs--------------------------------------------------------------------
1.模型介绍：模型使用resnet101架构

2.数据处理：
  自定义extract_frame()函数对具有不同帧数的姿态序列进行抽帧，统一帧数；
  使用cutout方法，对数据进行增强，与原始数据混合作为新的训练集

3.文件介绍：
  ①tool文件夹中存放数据读取文件dataset.py、数据增强文件cutout.py、
  模型定义文件model.py、数据可视化文件graph.py、visualise.py
	   
  ②data文件夹中存放训练与数据，若要修改训练数据路径，请于tool/dataset.py文件中修改，
  若要修改测试数据路径，请于test.py文件中修改

  ③model文件夹中存放模型文件，其中resnet101-5d3b4d8f.pth为未经训练的原始模型，
  net_trained为训练好的模型，压缩包为训练出的准确率为85%和86%模型，可进行替换。
  使用前应保证resnet101-5d3b4d8f.pth模型存放于

4.使用方法：
  直接运行train.py文件即可对模型开始训练，训练后的模型储存于model/net_trained.pth;
  直接运行test.py文件，即可获得测试集样本的平均识别准确率；
  直接运行show.py文件，可将数据可视化；
  注意：运行train.py后，模型会重新保存，请将model文件夹内的85%.zip解压后，覆盖原
  net_trained.pth后，再运行test.py文件，防止模型改变后准确率改变
